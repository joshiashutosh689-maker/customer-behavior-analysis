# 📊 Customer Behavior Analysis (Python + SQL + Power BI)

## 🚀 Project Overview
This project focuses on analyzing customer purchasing behavior to uncover key insights that drive business decisions.  
The workflow covers **data cleaning, analysis, and visualization**, transforming raw data into actionable insights.

---

## 🎯 Objectives
- Understand customer demographics and purchasing patterns  
- Identify top-performing product categories  
- Analyze the impact of discounts and subscriptions  
- Discover revenue trends across different segments  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy) – Data Cleaning & Preprocessing  
- SQL (MySQL) – Data Analysis & Querying  
- Power BI – Data Visualization & Dashboarding  

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning (Python)
- Handled missing values and inconsistencies  
- Removed duplicates and standardized column names  
- Performed feature engineering (e.g., age groups)

### 2️⃣ Data Analysis (SQL)
- Used aggregate functions (SUM, AVG, COUNT)  
- Applied GROUP BY, WHERE, HAVING for segmentation  
- Performed advanced analysis (ranking, comparisons, percentages)

### 3️⃣ Data Visualization (Power BI)
- Built an interactive dashboard with KPIs and filters  
- Visualized customer demographics, revenue trends, and behavior patterns  
- Added slicers for dynamic exploration

## 🔍 Key Insights
- Total revenue of **233K** from **3.9K customers** with an average purchase value of **~59.76**  
- **Middle-aged customers** are the primary revenue contributors  
- **Male customers (~67%) dominate** the customer base  
- **Clothing category** generates the highest sales  
- Sales remain **consistent across seasons**, indicating stable demand  
- Majority of purchases (**~57%**) occur **without discounts**  
- **Low subscription adoption (~27%)**, highlighting retention opportunity  

---

## 🚀 Business Recommendations
- Target **middle-aged customers** for higher conversions  
- Expand into **female segment** for growth opportunities  
- Improve **subscription/loyalty programs** to increase retention  
- Optimize **discount strategies** to maintain profitability  
- Enhance performance of **low-selling categories**
